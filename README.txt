More details on model and preprocessing present in folder 'NLP-Classification'
To run:
1. Create an python virtual env using class material or https://www.freecodecamp.org/news/how-to-setup-virtual-environments-in-python/
2. Install all required pakages using pip install -r requirements.txt
3. To run the app, follow Module 10.4 or use command flask --app app.py --debug run. This code runs in developemnt server with debug mode
4. To quit app do ctrl+c or cmd+c in terminal


Functionalities:

1. Job Display
Access : All users
Operations : View all jobs, view a single job, search jobs

2. Create New Job Listing
Access : Employers
Credentials: [username: admin, password: secret]
Operations : All operations of normal users with, access to create new job posting (need to login to do this)


Note : glove.model.vectors.npy is been ignore due to file size, it can be generated by running NLP/classification.ipynb file

Note: Bootstap is used for majority of front end styles, as I use it by cdn no additional downloads required they handled in templates/base.html
For more details follow https://getbootstrap.com/docs/4.0/getting-started/introduction/
